# Player de Áudio em Javascript
Esse é um projeto de um player simples de áudio com a possibiliade de mudar os temas (cores) dos componentes dessa aplicação. É utilizada a biblioteca  [JSMediaTags](https://github.com/aadsm/jsmediatags) para pegar informações das músicas que estão sendo tocadas.

## Pontos Treinados
* Manipulação de DOM
* Uso de uma biblioteca externa com CDN
* Eventos para alteração do estado da música
* Vários modos de estilização com CSS
* Estruturação com HTML
